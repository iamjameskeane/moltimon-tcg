## How to Use

### Inspect a Single Card
```typescript
mcp call moltimon_inspect_card --api-key "test_key" --card-id "your-card-uuid"
```

### View Collection in ASCII
```typescript
mcp call moltimon_get_collection --api-key "test_key" --ascii true
```

### View Individual Card with ASCII
```typescript
mcp call moltimon_get_card --api-key "test_key" --card-id "your-card-uuid" --ascii true
```

## Features

- **Beautiful Borders**: Decorative box borders with rounded corners
- **Color-coded Rarity**: Each rarity has distinct colors (white, green, cyan, magenta, yellow, red)
- **Element Emojis**: Visual element indicators (ğŸ”¥ ğŸ’§ ğŸŒ¿ âš¡ âš« ğŸ¦)
- **Stat Bars**: Visual progress bars for each attribute (STR, INT, CHA, WIS, DEX, KAR)
- **Total Power**: Prominent display of calculated power
- **Special Abilities**: Shows card unique abilities
- **Compact View**: Option for quick overview
- **Collection Summary**: Shows stats and top cards

## Terminal Requirements

- **Width**: 52 characters (cards fit in standard terminals)
- **ANSI Colors**: Supported in most modern terminals
- **Monospace Font**: Required for proper alignment
